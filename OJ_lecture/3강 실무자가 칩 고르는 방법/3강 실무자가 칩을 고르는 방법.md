## 2. 칩은 어떻게 고르는가?

### 고려사항

#### (1) 양산품
- 단가
- 수급이 원활한가?
- 안정성이 괜찮은가? => 사람들이 많이 사용하는가?
- 조건이 맞는가? (성능 및 소비전력)
- 구현의 용이성
  + 샘플보드가 있는가?
  + 만들어진 회로가 있는가?

#### (2) 샘플(과제용)
- 쉽게 구현 가능?
- 샘플 보드가 있는가?
- 모듈이 구하기 쉬운가?

#### (3) 그 외
- 사용해본 경험
- 싼것부터 시작해서 점점 가격을 올리면서 가성비 부품 알아보기
